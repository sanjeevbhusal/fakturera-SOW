A stripped version of a full stack application for invoice management. 

Tools used. 
- React JS for Frontend (Version: 19.1.1). 
- React Router DOM for Routing (Version: 19.1.1).
- Node Js with Express for Backend (Version: 24.4.1).
- Express JS for creating api routes (Version: 5.1.0)
- PostgreSQL for Database (postgres:15-alpine docker image for local development).
- Drizzle ORM for Database (Version: 0.44.7)


For Deployment, 
- Deployed on GCP Compute Engine i.e. Virtual Machine.
- Both Frontend and Backend on same machine.
- Used Apache to serve production build of Frontend. 
- Used Supabase to host Database.
